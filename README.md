# 🛒 E-commerce Sales Analysis using SQL & Power BI

## 📖 Project Overview

This project analyzes E-commerce sales data using **MySQL** and **Power BI** to generate business insights. It demonstrates SQL querying, data analysis, dashboard creation, and data visualization skills.

---

## 🎯 Objectives

- Analyze sales performance
- Identify top-selling products
- Find high-value customers
- Compare revenue across states
- Analyze payment methods
- Build an interactive Power BI dashboard
- ## 🛠️ Tools & Technologies

- MySQL
- SQL
- Power BI
- Microsoft Excel
- ## 📂 Dataset

The project uses the following datasets:

- Customers.csv
- Orders.csv
- Products.csv
- Categories.csv
- Payments.csv

## 💡 Key Business Insights

- Electronics generated the highest revenue.
- Maharashtra contributed the highest total revenue.
- UPI was the most preferred payment method.
- Premium Table 27 was the top-selling product.
- The dashboard highlights monthly sales trends and regional performance.

## 🚀 How to Run This Project

1. Import the CSV files into MySQL.
2. Execute the SQL scripts in order.
3. Open the Power BI `.pbix` file.
4. Refresh the data connection if required.
5. Explore the interactive dashboard using the available filters.
